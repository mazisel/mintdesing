📋 Nasıl Kullanılır?

  1. Yukarıdaki BACKEND_URL değişkenini canlı sunucu adresiniz ile değiştirin
  2. Bu HTML dosyasını tarayıcıda açın
  3. "Backend Versiyonu Kontrol Et" butonuna tıklayın
  4. Bir quote ID girin ve "Swiss QR Kodunu Test Et" tıklayın
  5. Token'ı girin (localStorage'dan alın: F12 → Application → Local Storage → token)
  6. Gelen QR kodun ortasında siyah haç olup olmadığını kontrol edin

Token Nasıl Alınır?

  1. Canlı sitenizde F12 basın
  2. Console sekmesine gidin
  3. Şunu yazın: localStorage.getItem('token')
  4. Enter'a basın
  5. Çıkan değeri kopyalayın
